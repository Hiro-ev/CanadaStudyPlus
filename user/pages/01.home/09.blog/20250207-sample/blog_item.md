# 新しいブログシステムでできること

このページは、ブログの新機能を紹介するサンプルページです。Markdown形式で記事を書き、画像を自由に配置できるようになりました。

## 1. 基本的な画像の挿入

通常の画像挿入は、以下のように記述します：

```markdown
![画像の説明](image.jpg)
```

実際の表示例：

![カナダの美しい風景](sample-image.jpg)

画像は自動的に中央に配置され、適切なサイズで表示されます。影付きの美しいデザインが適用されます。

## 2. テキストの回り込み配置

### 左寄せ画像の例

![left カナダ留学](sample-image.jpg)

画像を左に配置して、テキストを右側に回り込ませることができます。alt属性に「left」を含めるだけです。

```markdown
![left 説明文](image.jpg)
```

この機能により、雑誌のようなレイアウトが可能になります。長い文章と画像を組み合わせて、読みやすく視覚的に魅力的な記事を作成できます。カナダは世界中から留学生を受け入れており、質の高い教育と安全な環境で知られています。トロント、バンクーバー、モントリオールなど、各都市には特色があり、それぞれ異なる魅力を持っています。

### 右寄せ画像の例

![right 学生生活](sample-image.jpg)

同様に、画像を右に配置することも可能です：

```markdown
![right 説明文](image.jpg)
```

右寄せの画像は、記事に変化を与え、読者の注意を引きつけます。テキストと画像のバランスを考えながら、効果的に配置しましょう。カナダの教育機関では、実践的な学習プログラムが充実しており、インターンシップやCo-opプログラムを通じて、実際の職場経験を積むことができます。これらの経験は、卒業後のキャリア形成に大きく役立ちます。

## 3. 画像サイズの調整

### 小さい画像（50%幅）

![small コンパクトな表示](sample-image.jpg)

```markdown
![small ロゴや小さい画像](image.jpg)
```

小さいサイズの画像は、アクセントとして使用したり、複数の画像を並べて表示したい場合に便利です。

### 大きい画像（100%幅）

![large 迫力のある大画像](sample-image.jpg)

```markdown
![large パノラマや重要な画像](image.jpg)
```

大きな画像は、インパクトのあるビジュアルを提供し、読者の注目を集めます。

## 4. その他のMarkdown機能

### リスト表示

**番号付きリスト：**
1. 第一のポイント
2. 第二のポイント
3. 第三のポイント

**箇条書きリスト：**
- カナダ留学の魅力
- 充実したサポート体制
- 多様な教育プログラム

### 引用文

> 「このブログシステムにより、より魅力的なコンテンツを作成できるようになりました。画像とテキストの組み合わせが自由自在です。」

### テーブル

| 機能 | 説明 | 使い方 |
|------|------|--------|
| 左寄せ | 画像を左に配置 | `![left 説明](image.jpg)` |
| 右寄せ | 画像を右に配置 | `![right 説明](image.jpg)` |
| 小サイズ | 画像を50%幅で表示 | `![small 説明](image.jpg)` |
| 大サイズ | 画像を100%幅で表示 | `![large 説明](image.jpg)` |

## 5. 実際の記事作成手順

1. **記事用のフォルダを作成**
   ```
   user/pages/01.home/09.blog/YYYYMMDD/
   ```

2. **blog_item.md ファイルを作成**
   - タイトル、日付、タグなどのメタ情報を設定
   - Markdownで記事本文を作成

3. **画像ファイルを同じフォルダに配置**
   - JPG、PNG、GIF、WEBP形式に対応
   - ファイル名は記事内で参照

4. **プレビューで確認**
   - ブラウザで表示を確認
   - 必要に応じて調整

## まとめ

この新しいブログシステムにより：

✅ **柔軟な画像配置** - 左寄せ、右寄せ、中央配置が自由自在  
✅ **レスポンシブデザイン** - スマートフォンでも美しく表示  
✅ **簡単な記述** - Markdownの標準記法で記述可能  
✅ **プロフェッショナルな見た目** - 洗練されたデザインを自動適用  

ぜひこの新機能を活用して、魅力的なブログ記事を作成してください！
